# BOOK-SHEVKVES

#### 我的书架管理应用
## 目的
- 方便个人实体书架管理，让下线实体书籍变成数字化管理方式，可以通过图书扫码，拍照等方式将实体书籍录入应用中。

## 功能
1. 添加书籍
    - 扫条形码识别图书
    - 拍照图书封面
    - 修改图书内容
      - 名称
      - 出版社
      - ...
    - 购买时间
    - 标签设置
2. 我的书架列表
   - 按照时间排序
   - 按照标签筛选
   - 搜索关键字
3. 删除图书
4. 我的管理
   - 用户名称
   - 用户头像
   - ...
5. 分享我的书架
   - 图片分享我的书架
6. 我的标签管理
   - 创建标签
   - 修改标签
   - 删除标签

### 运行平台
微信小程序 
### 前端
taro + vue3
### 后端
nodejs + 微信云函数
### 三方功能
- 识别条形码api
- 豆瓣书籍评分等信息api
